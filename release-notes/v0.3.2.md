## Phosphene Electron v0.3.2

Patch release for Web Publish board image cache busting. Republishing a workspace now gives changed board PNGs a content-versioned URL, preventing browsers and CDNs from reusing an older board image after the page has been updated.

### Changes
- **Web Publish board assets** - names published board PNGs with a short SHA-256 content hash, for example `board-id-abcdef123456.png`.
- **Workspace manifests** - stores the hashed board image filename in `workspace.json` so generated pages reference the versioned asset normally.
- **Republish behavior** - removes stale board assets from the published site when a newer board image replaces them.
- **Regression coverage** - adds tests for changed image bytes across republishes, failed-publish cleanup, hashed site generation, and existing image hydration behavior.
