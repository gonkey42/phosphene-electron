## Phosphene Electron v0.4.0

Minor release for safer workspace cleanup and a more focused canvas workspace. This release adds two-stage deletes for destructive workspace and board actions, tightens the workspace chrome, and lets side panels collapse fully without losing browser or board context.

### Changes
- **Safe deletes** - adds two-stage confirmation for workspace and board deletes, with cancellation on Escape, blur, selection changes, workspace switches, and stale async completions.
- **Workspace tabs** - replaces wide publish/delete text actions with compact controls that stay reachable across long workspace names and crowded tab bars.
- **Board list** - tightens board rows for denser scanning while preserving rename, timestamp, publish, and delete reachability.
- **Collapsible side panels** - lets the boards and browser panels collapse to zero width while preserving saved layout state and live browser history.
- **Canvas focus controls** - adds bottom-right `< [] >` controls for hiding boards, hiding browser, or toggling a focused canvas view.
- **Regression coverage** - adds E2E coverage for compact workspace controls, board row density, safe delete behavior, panel collapse, browser-state preservation, and canvas controls across empty, loading, error, and loaded states.
