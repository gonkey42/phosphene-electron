## Phosphene Electron v0.2.7

Board-pack incremental import release. This version lets automation append generated board packs into existing workspaces by id, exact name, or the saved active workspace, while preserving the default behavior of creating a new workspace when no target is provided.

### Changes
- **Board pack targeting** — imports can now target an existing workspace by workspace id, exact workspace name, or active workspace.
- **Repeated imports** — targeted imports append boards to the selected workspace and make it active without renaming or re-iconing the workspace from the pack manifest.
- **CLI automation** — adds `--target-workspace-id`, `--target-workspace-name`, and `--target-active-workspace` to `npm run board-pack:import`, with strict malformed-argument rejection.
- **IPC and preload API** — exposes the same target selectors through `desktop.boardPacks.importFolder(packDir, options)` for future UI and automation paths.
- **Import safety** — validates targets before writes and keeps existing target workspaces safe during failed-import cleanup.
- **Documentation** — documents repeated target-import workflows and exact name matching behavior.
