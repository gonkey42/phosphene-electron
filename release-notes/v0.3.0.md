## Phosphene Electron v0.3.0

Web Publish dark-mode release. Published workspace sites now use Phosphene's app dark theme by default, and board snapshots render for dark viewing while preserving intentional board background colors.

### Changes
- **Web publishing** - generates index and workspace pages with Phosphene dark theme tokens, including dark workspace cards, board cards, image frames, links, and empty states.
- **Board snapshots** - exports Web Publish board previews through Excalidraw's dark theme path, using the app dark background only when a board has no explicit canvas background.
- **Theme safety** - adds token drift tests that compare Web Publish dark token mirrors against `src/App.css`.
- **Documentation** - documents that generated Web Publish pages are always dark and do not follow the app menu or viewer OS appearance setting.
