## Phosphene Electron v0.2.4

Security and stability patch release. This version removes raw renderer escape hatches, bounds workspace mounting so long-lived sessions stay leaner, and ships the related cleanup that landed with the merged remediation PR.

### Changes
- **Renderer bridge hardening** — replaces raw renderer SQL and filesystem escape hatches with task-scoped Electron bridge APIs for boards, workspaces, settings, theme, and storage access.
- **Workspace lifecycle fixes** — bounds mounted workspaces to the active neighborhood plus the current exit transition so previously visited workspaces do not stay mounted indefinitely.
- **Migration and verification coverage** — keeps migration, persistence, theme, and smoke coverage green across the narrowed bridge surface and release packaging flow.
- **Frontend cleanup** — removes unused frontend scaffolding, starter assets, and dead Excalidraw helper code that no longer contributes to the shipped app.
