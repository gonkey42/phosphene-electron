## Phosphene Electron v0.2.8

Web publishing release. This version adds manual private publishing for selected workspaces, generating static board snapshots and deploying them to a Cloudflare Access-protected Phosphene site.

### Changes
- **Workspace web publishing** — adds publish, republish, and unpublish controls for individual workspaces, with status labels for local-only, online, changed, and failed publish states.
- **Static snapshot export** — renders every board in a published workspace as PNG snapshots with filesystem image rehydration before export.
- **Private Cloudflare site** — generates a static workspace index and per-workspace pages for deployment to the separate `phosphene` Cloudflare Pages project at `phosphene.gonkey.org`.
- **Publish safety** — detects stale workspace changes during export, preserves the last successful site when deployment fails, and serializes publish, unpublish, and delete mutations.
- **Local artifact tracking** — records published workspace slugs, fingerprints, deployment URLs, and retryable failures in a filesystem-backed web publish manifest.
- **Documentation** — documents the Cloudflare setup, privacy model, publish behavior, and troubleshooting flow.
