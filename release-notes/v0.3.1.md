## Phosphene Electron v0.3.1

Patch release for Web Publish dark board snapshot exports. Published workspace pages are already dark; this release fixes light-authored Excalidraw boards so their published PNG previews render through Excalidraw's dark export path.

### Changes
- **Web Publish snapshots** - forces `exportWithDarkMode` for light or missing board canvas backgrounds during publish export.
- **Light-authored boards** - preserves `viewBackgroundColor: "#ffffff"` while enabling Excalidraw dark export conversion.
- **Dark-native boards** - avoids forcing dark export for explicit dark canvas backgrounds, preventing double inversion.
- **Regression coverage** - adds tests for saved white backgrounds, missing backgrounds, dark explicit backgrounds, and image hydration.

### Validation
