## Phosphene Electron v0.2.9

Web publish launch-environment fix. This patch lets Phosphene find Wrangler's `npx` command when the app is launched from Finder, Dock, or Spotlight instead of a terminal.

### Changes
- **Web publishing** — adds common macOS Node binary locations to the Wrangler deploy environment so GUI-launched app sessions can publish to Cloudflare Pages.

### Validation
- `npm test` — all tests passing
- `npm run lint -- --max-warnings=0` — zero warnings
- `npm run build` + `npm run build:main` + `npm run build:electron` — all succeed
- `npm run test:e2e` — smoke tests passing
