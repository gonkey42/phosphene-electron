# Phosphene

Phosphene is a React and Electron desktop canvas app built on Excalidraw with SQLite persistence,
filesystem-backed image extraction, and daily database backups.

## Development

- `npm install`
- `npm run dev`

## Verification

- `npm test`
- `npx tsc --noEmit`
- `npm run build`
- `npm run build:main`
- `npm run build:electron`
